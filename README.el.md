# NeoEthos

### *Ένα νέο ήθος για το trading — πειθαρχία θεσμικού επιπέδου, στα χέρια ενός ανθρώπου.*

**Καθαρή-Rust νοημοσύνη για trading: ανακαλύπτει μόνη της τις στρατηγικές της, μεγεθύνει κάθε θέση με τα μαθηματικά της επιβίωσης, και λογοδοτεί σε έναν και μόνο, ειλικρινή στόχο που ορίζεις εσύ.**

`100% Rust στο hot path` · `εγγενές cTrader` · `καθόλου Python στο runtime` · `το μηχάνημά σου, τα δεδομένα σου, τα κλειδιά σου`

## Γιατί υπάρχει

Τα σοβαρά εργαλεία — walk-forward validation, γενετική αναζήτηση στρατηγικών, sizing με λογική Kelly, όρια ρίσκου prop-firm, πραγματικό ensemble μοντέλων — ζούσαν πάντα πίσω από θεσμικούς τοίχους και πεντάψηφες συνδρομές. Ο μικρός trader παίρνει ένα chart και μια προσευχή.

Το NeoEthos είναι η άρνηση αυτού. Βάζει *ολόκληρη την αλυσίδα* — έρευνα, ανακάλυψη, ρίσκο, εκτέλεση — στο laptop ενός ανθρώπου, σε λογισμικό ελέγξιμο γραμμή-γραμμή που δεν λέει ψέματα σε κανέναν: κάθε αριθμός βγαίνει από τη μηχανή· τίποτα δεν εφευρίσκεται για το UI.

## Τι κάνει

- **Ανακαλύπτει στρατηγικές, δεν πουλά εικασίες.** Μια γενετική αναζήτηση γεννά και δοκιμάζει στρατηγικές σε πολλαπλά timeframes, και μετά τις περνά από **πέντε υποχρεωτικές πύλες κατά της υπερπροσαρμογής** — walk-forward, CPCV, PBO/CSCV (López de Prado), permutation test και parameter-plateau test — ώστε ό,τι φτάνει σε σένα να έχει αντέξει σε δεδομένα που δεν είδε ποτέ στην εκπαίδευση. Δες το [PRINCIPLES.md](PRINCIPLES.md).
- **Μοιράζεται υπολογιστική ισχύ, όχι μυστικά.** Το **Federation** (λογική SETI@home): άνθρωποι που εμπιστεύονται ο ένας τον άλλον ενώνουν τα μηχανήματά τους για την ανακάλυψη — χωρίς κανέναν server, με κάθε εισαγόμενο αποτέλεσμα να ξαναπερνά όλες τις τοπικές πύλες πριν αγγίξει αληθινά χρήματα. Δες τον [οδηγό Federation](docs/FEDERATION.md)· το serverless P2P mesh είναι το απομονωμένο sidecar [`mesh/`](mesh/README.md).
- **Δύο ειλικρινείς λειτουργίες, ένας κεντρικός διακόπτης.** Εσύ διαλέγεις τον στόχο· *η αναζήτηση, τα μοντέλα και το ρίσκο αναπροσανατολίζονται όλα γύρω του.*
- **Μεγεθύνει για επιβίωση.** Το μέγεθος θέσης είναι κλάσμα του *ζωντανού* υπολοίπου σου (ανατοκίζεται καθώς μεγαλώνεις), παραγόμενο από το μετρημένο πλεονέκτημα κάθε στρατηγικής και τα πραγματικά ανά-lot κόστη του broker — όχι ένα σταθερό, καταστροφικό ποσοστό.
- **Πραγματικό ensemble μοντέλων.** XGBoost, CatBoost, LightGBM, νευρωνικά δίκτυα (Burn), KAN, N-BEATS, TabNet, TiDE και άλλα — εγγενής Rust, χωρίς GIL, χωρίς ενσωματωμένη Python στο hot path.
- **Live, εγγενώς στον broker.** cTrader Open API πάνω από OAuth/WebSocket. Εντολές αγοράς **και** υπό όρους (limit/stop), ζωντανό P/L, ημερολόγιο συναλλαγών τύπου MyFxbook. Ο λογαριασμός σου, τα κλειδιά σου, στο μηχάνημά σου.
- **Desktop εφαρμογή *και* τερματικό.** Μια μονο-διεργασιακή desktop εφαρμογή **Tauri + React** (Ελληνικά / Αγγλικά) και ένα πλήρες ratatui TUI με ζωντανά candlesticks — η ίδια Rust μηχανή από κάτω.

## Οι δύο λειτουργίες

| | **Risky Mode** | **Prop-Firm Mode** |
|---|---|---|
| **Στόχος** | Πολλαπλασιασμός ενός μικρού υπολοίπου προς έναν μεγάλο στόχο, όσο γρήγορα επιτρέπει το πλεονέκτημα | Άνετο πέρασμα των challenges μιας prop-firm και σταθερή μηνιαία απόδοση |
| **Εσύ ορίζεις** | αρχή → στόχο → ορίζοντα (π.χ. €100 → €50.000 σε 6 μήνες) | το preset της εταιρείας (FTMO, FundedNext, …) + όρια drawdown |
| **Η αναζήτηση** | *πιέζεται* να βρει στρατηγικές που μπορούν να φτάσουν τον στόχο σου στην ώρα του | βελτιστοποιεί για τους κανόνες παραθύρου της εταιρείας |
| **Η αλήθεια** | υψηλό ρίσκο, βαθύ drawdown — και οι πιθανότητες υπολογίζονται, δεν υπόσχονται | πρώτα η ασφάλεια και η σταθερότητα |

Εσύ ορίζεις τον στόχο. Τα μαθηματικά σου λένε την αλήθεια γι' αυτόν. Το bot προσπαθεί.

> **Προειδοποίηση κινδύνου.** Το trading σε μοχλευμένο FX ενέχει ουσιώδη κίνδυνο απώλειας και δεν είναι κατάλληλο για όλους. Το NeoEthos είναι λογισμικό έρευνας/εκπαίδευσης και **όχι επενδυτική συμβουλή**. Τίποτα εδώ δεν είναι υπόσχεση κέρδους. Χρησιμοποίησε demo λογαριασμό μέχρι *εσύ* να επαληθεύσεις ένα πλεονέκτημα· εσύ και μόνο ευθύνεσαι για κάθε live trading.

## Αρχιτεκτονική

Πλήρως καθαρή Rust — μεταναστεύσαμε από ένα υβρίδιο Python/Rust για να πεθάνει το GIL και να κερδίσουμε memory safety από άκρη σε άκρη. Μία και μόνη διεργασία Tauri συνδέει τα crates της μηχανής **in-process** και σερβίρει το React UI· δεν υπάρχει ξεχωριστό backend να τρέξεις.

- **`neoethos-core`** — διαχείριση ρίσκου, βελτιστοποίηση χαρτοφυλακίου, το ενιαίο config που επεξεργάζονται UI/TUI
- **`neoethos-data`** — υψηλής ταχύτητας OHLCV + μηχανή χαρακτηριστικών, μηδενικό look-ahead bias
- **`neoethos-models`** — το εγγενές ML ensemble
- **`neoethos-search`** — γενετική ανακάλυψη + κατάταξη με επίγνωση στόχου + οι πύλες OOS validation
- **`neoethos-trader`** — παραγωγή ζωντανών σημάτων + η αυτόνομη μηχανή
- **`neoethos-app`** — in-process HTTP API, headless εργασίες, μεταφορείς cTrader
- **`neoethos-cli`** — το TUI + batch εργασίες χειριστή
- **`desktop/`** — το κέλυφος Tauri v2 + React/TypeScript UI (`crate neoethos-desktop`)

## Ξεκινώντας

**Θες απλώς να το τρέξεις;** Κατέβασε τον Windows installer (`.exe` ή `.msi`) από την [τελευταία έκδοση](../../releases/latest). Άνοιξέ τον, σύνδεσε τον cTrader λογαριασμό σου (OAuth, μία φορά), και μπήκες. Χρησιμοποίησε πρώτα **Demo** λογαριασμό.

**Θες να το χτίσεις μόνος σου;** Δες το **[BUILDING.md](BUILDING.md)** για τον πλήρη οδηγό ανά λειτουργικό (toolchains, το desktop build, το TUI, προαιρετική επιτάχυνση GPU, και ρύθμιση cTrader). Η σύντομη εκδοχή:

```bash
# Προαπαιτούμενα: Rust (stable, edition 2024) + Node 20+ + τα Tauri OS deps.
# Installer της desktop εφαρμογής (πακετάρει frontend + μηχανή σε ένα binary):
cd desktop
npm install
npx tauri build          # → target/release/bundle/{nsis,msi}/...

# Ή τρέξε τη ζωντανά κατά την ανάπτυξη:
npx tauri dev

# Terminal UI (ζωντανά candlesticks, ανακάλυψη, logs) — δεν χρειάζεται Node:
cargo run --release -p neoethos-cli
```

## Κατάσταση

**v0.5.6.** Όλα όσα έφερε η v0.5.2 — «the honesty release» (η στοίβα validation πέντε πυλών: walk-forward, CPCV, PBO, permutation, plateau· risk-constrained Kelly sizing πάνω στην εμπειρική κατανομή συναλλαγών· ο προσομοιωτής challenge prop-firm· auto-cull με μόνιμη blacklist + αυτόματη επανα-ανακάλυψη· Federation Phase 0) — και επιπλέον, έκτοτε: το **serverless P2P mesh sidecar είναι πραγματικότητα** ([`mesh/`](mesh/README.md) — αυτόματη συνδεσιμότητα μέσω relay, ανακάλυψη peers με gossip, κατανομή εργασιών ανακάλυψης μεταξύ μηχανημάτων, συνάθροιση δυναμικότητας του swarm), ένα απομονωμένο **MCP tool sidecar** για τον AI supervisor, **hold-to-bracket ισοτιμία live εκτέλεσης** με τον επικυρωμένο πυρήνα ανακάλυψης, never-OOM holdout views, και ένας **γραμμή-γραμμή έλεγχος όλου του πρωτογενούς κώδικα** (Ιούλιος 2026). Ο δρόμος μπροστά: κατανεμημένη ανακάλυψη island-model (σε δοκιμές σε δύο μηχανήματα) → adapters ανεξάρτητοι broker → mobile monitor.
